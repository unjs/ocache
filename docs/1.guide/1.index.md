# Getting Started

ocache is a set of standalone caching utilities. It has zero framework dependencies and works with any runtime that has standard `Request`/`Response`.

It gives you two primary tools:

- [`defineCachedFunction`](/guide/functions) — wrap any async function with TTL, stale-while-revalidate, integrity checks, and in-flight request deduplication.
- [`defineCachedHandler`](/guide/handler) — wrap an HTTP handler with full response caching (`etag`, `last-modified`, `304 Not Modified`, `Cache-Control`, and more).

## Quick Start

Wrap any function with `defineCachedFunction` to add caching with TTL, stale-while-revalidate, and request deduplication:

```ts
import { defineCachedFunction } from "ocache";

const cachedFetch = defineCachedFunction(
  async (url: string) => {
    const res = await fetch(url);
    return res.json();
  },
  {
    maxAge: 60, // Cache for 60 seconds
    name: "api-fetch",
  },
);

// First call runs the function; subsequent calls return the cached result.
const data = await cachedFetch("https://api.example.com/data");
```

> [!TIP]
> All durations (`maxAge`, `staleMaxAge`, TTL) are in **seconds**.

By default, ocache stores entries in an in-memory `Map`. Swap in Redis or any other backend via [custom storage](/guide/storage).
